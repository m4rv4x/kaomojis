# kaomojis (◕‿◕✿)

Curated collection of Japanese kaomoji emoticons — ready to copy-paste into chats, terminals, and social media.

## What's inside

- **Happy**: `(◕‿◕✿)` `(ﾉ◕ヮ◕)ﾉ*:･ﾟ✧` `(◠‿◠✿)`
- **Sad**: `(╥﹏╥)` `(｡•́︿•̀｡)` `(╯︵╰,)`
- **Angry**: `(╬ Ò﹏Ó)` `(≧□≦)` `(◣_◢)`
- **Surprised**: `(⊙_⊙)` `(°ロ°)` `∑(O_O;)`
- **Love**: `(♡˙︶˙♡)` `(｡♥‿♥｡)` `(´▽`ʃ♡ƪ)` 
- **Table flip**: `(╯°□°)╯︵ ┻━┻`
- **Shrug**: `¯\_(ツ)_/¯`

## Usage

Browse the collection and copy any kaomoji into your clipboard. Works anywhere that accepts Unicode text.
